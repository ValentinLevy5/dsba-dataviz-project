# Media Lens – US Political News Coverage

Data Visualization course project exploring how America's top news outlets cover politics.

## Run locally

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## Data

Data sourced from the [GDELT Project](https://www.gdeltproject.org/) covering 7 major US news outlets across 6 political topics (2017-2025).

## Tech stack

- **Streamlit** – web app framework
- **Altair** – interactive visualizations
- **Pandas** – data processing
